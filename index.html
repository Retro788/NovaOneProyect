This project is part of the **NASA Space Apps** initiative, focusing on the development of a **Detection Model** through complex neural networks. The model is trained bifurcatively, utilizing an advanced detection pattern and mobility prediction via geometric topologies. This seismic detection system is integrated into a solar simulator, which allows for more intuitive and accessible visualization and comprehension for the end user, leveraging cutting-edge artificial intelligence and astronomical simulations.
